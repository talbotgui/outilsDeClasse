Bonjour et bienvenu dans l'application de gestion de votre classe.

Si vous accédez à cette application pour la première fois, voici quelques informations utiles :

  • L'aide est disponible, à tout moment, en cliquant sur le bouton présent en haut à droite de cet écran.
  • Quand aucune donnée de classe n'est chargée, sont disponibles les onglets Accueil, Charger mon année et Créer mon année.
  • Si vous avez déjà créé une année et que vous souhaitez charger les données précédemment renseignées, utiliser l'onglet Charger mon année pour reprendre votre travail.

Si vous souhaitez utiliser cette application sans connexion Internet, c'est possible. Pour cela, vous avez deux options :

  • 1/ Demande à votre navigateur de télécharger l'application
    • Cliquer sur le bouton "Installer outil de classe" affiché tout à droite dans votre barre d'adresse.
    • Un raccourci est alors créé sur votre bureau
    • Avantage : l'application se met à jour régulièrement
  • 2/ Télécharger, sur votre poste, le site et un petit serveur WEB :
    • Télécharger l'archive outils-de-classe_1.0.0.versionLocale.zip sur votre ordinateur
    • Ouvrir l'archive outils-de-classe_1.0.0.versionLocale.zip
    • Extraire le contenu dans un répertoire de votre choix
    • Double cliquer sur demarrerServeurLocal.cmd
    • L'application est maintenant disponible à l'adresse http://localhost:8000/outilsDeClasse
