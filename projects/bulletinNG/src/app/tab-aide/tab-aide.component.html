Aide
Cette page d'aide est accessible depuis le bouton ? situé en haut à droite de la page.
Liste des sujets traités :
  • Conservation du dernier fichier sauvegardé
  • Sauvegarder un fichier depuis le serveur de données
  • Chargement d'un fichier depuis le serveur de données
  • Expressions de recherche
Chargement d'un fichier depuis le serveur de données
Au clic sur le bouton de chargement de l'écran principale, l'application tente de lister les sauvegardes disponibles sur le serveur présent à l'adresse 192.168.1.52. Hors les navigateurs refusent de laisser s'exécuter cette requête car l'adresse demandée n'est pas celle du site affichant la page.
  • Sous Chrome, une icône du type s'affiche alors dans la barre d'adresse. Pour autoriser l'affichage de la liste des sauvegardes du serveur, il suffit de cliquer sur cette icone puis de cliquer sur le message 'Charger les scripts à risque'.
Sauvegarder un fichier depuis le serveur de données
Si, au clic sur le bouton "Sur le serveur" dans la popup de sauvegarde, apparait un message indiquant une erreur alors votre navigateur a certainement du tenter vous protéger.
  • Sous Chrome, une icône du type est certainement apparue dans la barre d'adresse. Surtout ne cliquer pas dessus car cela raffraichira la page et les données seront perdues. Utiliser d'abord la sauvegarde en local puis cliquer sur l'icone pour autoriser les scripts. Enfin, recharger les données locales depuis réaliser votre sauvegarde sur le serveur.
Conservation du dernier fichier sauvegardé
L'application est en mesure de se souvenir du nom de votre dernière sauvegarde et de vous afficher un avertissement si vous charger un fichier qui n'est pas le dernier modifié.
Mais pour cela, l'application doit être en mesure de stocker ce nom de fichier dans votre navigateur et ce dernier ne doit pas le détruire lors de sa fermeture.

Si cette fonctionnalité n'est pas opérante dans votre navigateur, suivez la procédure ci-dessous :
  • Chrome :
    • Saisir dans la barre d'adresse "chrome://settings/content/cookies"
    • En face de "autoriser", cliquer sur le bouton Ajouter et ajouter le site "https://me.guillaumetalbot.com/maclasse"
    • Pour les développeurs, ajouter aussi "http://localhost:4200"

    Si vous utiliser une extension de Chrome supprimant les données à la fermeture du navigateur, pensez à ne pas supprimer les "local storage data".
Expressions de recherche
Dans les recherches de textes (recherche de compétences dans l'onglet 'Compétences' et dans les composants de sélection de compétence), il est possible de rechercher un mot et/ou d'exclure un mot (en le préfixant d'un tiret).
Exemple : Multiplication -addition va rechercher les compétences contenant le mot "Multiplication" mais sans le mot "Addition"
