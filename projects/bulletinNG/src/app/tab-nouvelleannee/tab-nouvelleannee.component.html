Cette application ne stocke aucune donnée. C'est à vous, utilisateur, de charger et sauvegarder vos données dans un fichier que vous conservez sur votre ordinateur. Ces données sont donc sous votre responsabilité.

Après avoir rempli le formulaire ci-dessous, en cliquant sur le bouton Créer et charger les données, vous allez ouvrir l'application avec un fichier de données initial. Ce fichier de données comprend un élève, le socle commun 2017 des cycles 2, 3 et 4.
